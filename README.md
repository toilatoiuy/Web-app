## ✨ Điểm nổi bật
- **Giao diện quen thuộc:** Thiết kế chuẩn iOS giúp thao tác nhanh và chính xác.
- **Cài đặt như App thật:** Hỗ trợ PWA, xuất hiện biểu tượng trên màn hình chính.
- **Dùng không cần mạng:** Tự động lưu dữ liệu để chạy Offline 24/7.

## 🚀 Cách cài đặt nhanh
1. Truy cập: [Dán link Netlify của bạn vào đây]
2. **Trên Android:** Chọn dấu 3 chấm dọc ⋮ -> "Cài đặt ứng dụng".
3. **Trên iPhone:** Chọn nút Chia sẻ (hình vuông mũi tên lên) -> "Thêm vào MH chính".
